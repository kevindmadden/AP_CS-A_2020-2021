Project #02 - Draw Something

Description:

Create your own original drawing.

Requirements:

  1. Your drawing should have clear intention behind it. In other words, it should be obvious you were trying to create something specific.
  2. Experiment around with the below methods to create your drawing. While you do not need to use all of the below methods in your final drawing, please at least experiment around with all of them.
    • point
    • line
    • circle
    • ellipse
    • square
    • rectangle
    • filledCircle
    • filledEllipse
    • filledSquare
    • filledRectangle
    • arc
    • setPenRadius
    • setPenColor
    • setScale
  3. If I did not mention a method in the list above, you can ignore the description of the method in the documentation.

Reminder:

  • The purpose of the assignment is for you to become comfortable using methods. Please ask questions as they arise. I do realize that this is the first time many of you are using methods.
